# GeoFotoMap
## Progressive Web App _Przewodnik Przyrodniczy_ 


![icon](icons/icon-192.png)

### Jak działa?
1. Włączenie kamery i zrobienie zdjęcia z poziomu aplikacji
2. Pobranie GPS
3. Pojawienie się punktu na mapie
4. Udostępnienie zdjęcia i jego lokalizacji, dzięki Web Share API
5. Instalacja PWA
6. Uruchomienie offline


### Cechy aplikacji:
- użycie kamery na telefonie i laptopie 
- możliwość wyboru zdjęcia z pliku przez input type="file"
- geolokalizacja (Geolocation API)
- systemowe udostępnianie (Web Share API)
- mapa OpenStreetMap (Leaflet)
- instalacja jako PWA (plik manifestu)
- działanie offline (service worker)
- responsywny interfejs przy użyciu biblioteki Bootstrap i pliku CSS
